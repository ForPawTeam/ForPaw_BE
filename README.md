# 🤔 프로젝트 소개


## 개발 필요성

현재 많은 사람들이 반려동물을 키우고 있으며, 이에 따라 유기동물 문제가 점차 심각해지고 있고, 국가와 지자체 차원의 인프라와 시민들의 인식이 여전히 부족하여 많은 동물들이 적절한 보호를 받지 못하고 있습니다. 
<br/> 
<br/> 
부산의 경우, 매년 천여 마리의 동물이 버려지지만, 동물보호센터의 수는 매우 부족합니다. 또한, 보호소와 봉사자의 연결이 원활하지 않아 보호소는 인력 부족을 겪고 있으며, 봉사자들은 참여 기회가 제한적입니다.
<br/>
<br/> 
이에 저희 '포포' 팀은 유기동물 보호소와 봉사자를 효율적으로 연결하고, 유기동물 입양 문화를 활성화하기 위해 투명하고 신뢰할 수 있는 플랫폼을 기획하게 되었습니다! 
이를 통해 유기동물 보호와 입양을 촉진하고, 보호소와 봉사자 간의 원활한 소통을 지원하고자 합니다.
<br/>
<br/>
<br/>
<br/>

## 서비스 소개

1. 보호소는 필요한 봉사 일정을 등록하고, 개인 또는 모임은 이를 열람하여 신청할 수 있습니다.
2. 지역, 관심사 등을 기반으로 봉사 모임을 자유롭게 조직하여 활동할 수 있습니다.
3. 인증된 유기동물 보호소에서 유기동물을 입양하고 사후에 도움과 케어를 받을 수 있습니다.
4. 커뮤니티, 채팅 기능을 활용하여 보호소, 개인, 모임 간의 소통이 원활히 이루어질 수 있습니다.
5. 상호 니즈에 맞는 다양하고 장기적인 봉사를 진행할 수 있습니다.
